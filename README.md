# homewebapi 功能
1.采集家里面的环境参数接口：温度，湿度  
2.门口监控，家门前一直有小猫，爬来爬去，甚至爬到车上和凳子上，因此有个程序会抓取喵的图片通过mqtt传到服务端，另一个程序会通过接口或者存到数据库中或者文件夹（NAS）中，并且在数据库中备份  
3.身份认证接口：单点登录  
4.到家时间记录接口  
5.定时调用接口来告诉我当前时间和明天天气  
6.mqtt控制  
7.房间人体红外信号采集接口：mqtt  
## 数据库
mysql 5.5  
## 接口 类型
目前规划基本上就是webapi  
包含部分mqtt接口跳转
